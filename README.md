# 🤖 AURA AI TRACKER
Professional Expense and Habit Intelligence Dashboard.

## 🚀 Installation
1. Install Python 3.12+
2. Run `pip install -r requirements.txt`
3. Run `python main.py`

## ✨ Features
- **Smart Budgeting:** Track spent vs. funds.
- **Habit HUD:** Interactive cards with consistency scoring.
- **Alerts:** 10% budget warnings and habit notifications.
